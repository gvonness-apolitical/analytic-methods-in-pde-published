# Analytic Methods in Partial Differential Equations

**Author:** Gregory T. von Nessi

This repository contains the published PDF of *Analytic Methods in Partial Differential Equations*, a comprehensive textbook covering classical and modern approaches to PDE theory.

## Contents

- `analyticMethodsPDEv2011.pdf` - The complete textbook (2011 edition)

## Topics Covered

### Part I: Classical Approaches
- Physical motivations for PDEs
- Laplace's and Poisson's equations
- Heat equation
- Wave equation
- First-order PDEs
- Conservation laws
- Classification: elliptic, parabolic, and hyperbolic equations

### Part II: Functional Analytic Methods
- Functional analysis foundations
- Function spaces (Lebesgue, Sobolev)
- Linear elliptic PDE theory
- Distributions and Fourier transforms
- Variational methods

### Part III: Modern Methods for Fully Nonlinear Elliptic PDE
- Non-variational methods for nonlinear PDEs
